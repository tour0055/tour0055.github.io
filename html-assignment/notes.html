HTML Tags and Descriptions

Main:
Defines the primary content of the document.
Article:
Represents an independent piece of content, such as a blog post or news article.
Aside:
Defines content aside from the main content, such as a sidebar or related links.
img:
Shows a picture on a webpage using the "src" attribute to set the image source.
